# Support

## Getting Help
- Check `docs/START_HERE.md` first.
- Check `docs/TROUBLESHOOTING_DECISION_TREE.md` for common failures.
- Search existing issues before opening a new one.

## Bug Reports
Include:
- environment (OS, Python version, model provider)
- exact command or endpoint
- expected behavior
- actual behavior
- logs or stack trace

## Feature Requests
Describe:
- use case
- expected API or UX
- backward compatibility concerns
